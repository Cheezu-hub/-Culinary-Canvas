# Culinary Canvas

Culinary Canvas is a premium, beautifully designed recipe sharing platform. It features a stunning, dynamic user interface with glassmorphism, responsive design, and smooth animations. Users can discover, search, and upload their own recipes.

## Features
- **Discover Recipes**: Browse a global collection of recipes.
- **Search & Filter**: Find recipes by cuisine or dietary needs.
- **User Accounts**: Register and log in to save favorites.
- **Upload Recipes**: Share your own culinary creations with the community.
- **Premium UI**: Experience a dynamic 10/10 design with soft glows, glassmorphism, and custom animations.

## Tech Stack
- **Frontend**: Vanilla HTML/CSS/JavaScript, bundled with [Vite](https://vitejs.dev/).
- **Backend**: Node.js, Express, MongoDB.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- MongoDB running locally or a MongoDB URI

### Backend Setup
1. Navigate to the `backend` directory.
2. Run `npm install` to install dependencies.
3. Rename `.env.txt` to `.env` and fill in your configuration details (MongoDB URI, Cloudinary credentials, JWT Secret).
4. Run `npm run dev` or `node server.js` to start the backend server on port 5000.

### Frontend Setup
1. Navigate to the `frontend` directory.
2. Run `npm install` to install Vite and other dependencies.
3. Run `npm run dev` to start the Vite development server.
4. Open the provided localhost URL in your browser (typically `http://localhost:5173`).

## Design
The design focuses on dark aesthetics with vibrant orange and gold accents, using deep shadows and blurred backdrops to create a high-end, modern look.
